\##############

MICROBEXPLORER

\###############

Pipeline post-traitement qui permet de nettoyer, réduire et créer des graphiques sur des données issues de kaiju, mégahit ou bakta.



Ce que la pipeline gère :



Graphiques : Histogramme, ACP, Heatmap et dendrogramme

Test : Permanova

Traitement : CLR



L'application Shiny vous permet de visualiser et sauvegarder le style des graphiques pour l'appliquer directement à la pipeline.
